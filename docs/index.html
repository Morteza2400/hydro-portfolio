About this demo

This page demonstrates diameter-based symbology, popups with hydraulic attributes, and basic analytics fed by a CSV timeseries. In your repository, include a sql/ folder for schemas & queries.

  • Map: Leaflet + OSM basemap
  • Data: data/network.geojson and data/sensors.csv
  • Charts: Chart.js

How to use

  1. Click a pipe or node to view attributes (diameter, roughness, PRV setpoint).
  2. KPIs update from the loaded timeseries file.
  3. Clone the repo to view SQL and modelling notes.
